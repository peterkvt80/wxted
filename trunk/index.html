Welcome to wxTED teletext editor

Description

wxTED is a teletext editor. It is currently Windows only but with some help and resources it could be OSX and Linux too.
Basic editing is now implemented including carousels and language options. Yet to be implemented: Edit tools, more Publishing options, more languages.
There are are some sample pages bundled so you can get started with real pages as starting points.

Install

  1. Unpack the contents of the zip file into the folder that you want to run it from.
  2. Open teletext2.ttf by double clicking on it and press the Install button
  3. The program is wxTED.exe which can be run by double clicking on it.
  4. The instructions are in the accompanying PDF manual
